ls - показывает все что есть в директории (папки, файлы, фото, видео, музыка, зипки) в той директории где вы находитесь
ls -a  - показывает тоже самое что и ls, а также скрытые файлы и директории 
ls -l  - показывает более подробную инф. о папках и файлах
ls -la (ls -a -l)  -  показывает дпоп. инф. о папках и фалайх + доп. инф. о скрытых файлах и папках
ls путь до папки  - показывает содержимое папки не заходя в нее ls Desktop/pythonv1/base  - пример

cd  - переход в домашнюю папку
cd ~  - переход в домашнюю папку
cd name  - переход в папку name, если такая папка существует в текущей директории
cd .  - переходит в текущую директорию
cd ..  - переходит на одну папку назад
cd ../..  -переходит на две папки назад (можно на большее количество папок)

mkdir test  - создает директорию test в текущей папке
mkdir путь/test  - создает директорию test по указанному пути
mkdir test1 test2 test3  - создаст несколько папок: test1 test2 test3 в текущей папке

touch test.txt  - создание файла test с расширением txt

rm test  - удаляет файл
rmdir test  - удаляет папку, если она пустая
rm -rf test  - удаляет папку test, пустая или не пустая не имеет значения

mv test1.txt test2.txt  - переименовывает первую папку на вторую указанную
mv test1.txt dirtest  - переместит файл test1.txt в папку dirtest (если такая папка dirtest имеется, в ином случае test1.txt переименуется в dirtest)

history  - показывает историю всех введеных команд на данном терминале
history 10  -показывает последние 10 введеные команды

pwd (present/print work directory)  - показывает путь до той директории, где вы выполняли эту команду

nano test.txt  - откроет файл test.txt в текстовом редакторе nano, если такого файа нет то создаст его

cat test.txt  - выведет в терминале содержимое файла test.txt

brew  - системный установщик пакетов (macOs)
apt  - системный установщик пакетов
snap  - системный установщик пакетов (в основном для приложений)
sudo  - super user do
